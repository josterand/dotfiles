## Philosophy

This rice setup is custom-designed from the ground up by me to work seamlessly with the bspwm window manager. Its interface embraces a minimalist yet functional philosophy, featuring a single-bar setup that prioritizes essential information. This ensures users to focus on what truly matters. As my flagship rice, as long as I continue using bspwm, refinements, updates, new functionality, and fixes will keep coming!

## System

-   **Operating System**: Linux
-   **Window Manager**: bspwm (with the sxhkd keybind daemon)
-   **Terminal**: Alacritty
-   **Bar**: Polybar
-   **Launcher**: Rofi
-   **Notification Daemon**: Dunst
