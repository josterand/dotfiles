# About
This is the cursor that I use as a complement to my system setup. I didn't like Linux Mint's default cursor, so I looked for an alternative that was more compact, sharp, and comfortable to look at. So [Apple Cursors (macOS Cursors)](https://github.com/ful1e5/apple_cursor) was the right choice for me.

# Installation
Installation

- To install locally (current user): move the `macOS` folder and its contents to `~/.icons/`
- To install globally (for all users): move the `macOS` folder and its contents to `/usr/share/icons/` (admin rights `sudo` required)
